    GK360 API Documentation

    Introduction

    GK360 stands for GemeindeKompass360 which is German and means something like "Municipality Compass 360". Basically this API can give you all the data about Austrian Villages, Towns, Cities, Counties and Federal States.

    Free

    The API is free to use for everyone.

    No API key required

    Endpoints

    The API provides various endpoints to retrieve detailed information about municipalities, districts, and federal states.

    Municipalities

    The API provides the following endpoints to retrieve detailed information about municipalities:

    1. Get Information for a Specific Municipality by Code

    Endpoint: ?code={municipality}
    Description: Returns all data for a municipality, including contact information, statistics, and images. 

    
fetch('https://www.gk360.at/api/municipalities/?code=bregenz')
    .then(response => response.json())
    .then(data => {
        console.log(data);
    });

    2. Get Images for a Specific Municipality

    Endpoint: ?action=images&municipality={municipality_id}
    Description: Returns all images (Thumbnail, Full-Size, Attribution) associated with a specific municipality. 

    fetch('?action=images&municipality=5')
                .then(response => response.json())
                .then(data => {
                  console.log(data);
                });

    Example Response:

fetch('https://www.gk360.at/api/federal-states/?action=all')
