    GK360 API Documentation

    Introduction

    GK360 stands for "GemeindeKompass360," which translates to "Municipality Compass 360" in English. This API provides comprehensive data about Austrian villages, towns, cities, counties, and federal states.

    Free

    The API is completely free for everyone to use without any limitations.

    No API Key Required

    Access the API effortlesslyâ€”no API key is needed for any request.

    Comprehensive Data

    Access all information about municipalities, counties, and federal states through a single API.

    Easy-to-Use Endpoints

    Designed with simplicity in mind, the API endpoints are intuitive and user-friendly.

    Getting started

    This guide will get you all set up and ready to go with this documentation template. We will cover how to get started quickly and view a couple of customization techniques.

    Quickstart

    The default API url is https://www.gk360.at/api/*endpoint*/?*parameters*.

    Endpoints

    1. Municipalities
    2. Counties
    3. FederalStates
    4. Search

    https://www.gk360.at/api/*endpoint*/?*parameters*

    How does it work?

    The API collects data primarily through web scrapers. The following sites are scraped by this API:

    Source Used for
    https://commons.wikimedia.org Images
    https://geocode.maps.co Coordinates (Latitude, Longitude)
    https://de.wikipedia.org General Information
    https://gemeinde-osterreich.at General Information
    Google Gemini API Descriptions
    Google Place API Opening Hours

    Endpoints

    The API provides various endpoints to retrieve detailed information about municipalities, districts, and federal states.

    Municipalities

    The API provides the following endpoints to retrieve detailed information about municipalities:

    1. Get Information for a Specific Municipality by Code

    Endpoint: ?code={municipality}
    Description: Returns all data for a municipality, including contact information, statistics, and images. 

    
fetch('https://www.gk360.at/api/municipalities/?code=bregenz')
    .then(response => response.json())
    .then(data => {
        console.log(data);
    });

    Examples (Try them out)

    The following examples demonstrate how to use the API. Click on the links to see the API responses in new tabs.

    Municipalities

    Retrieve detailed data about specific municipalities. For example:

    https://www.gk360.at/api/municipalities/?code=bregenz
    https://www.gk360.at/api/municipalities/?code=dornbirn
    https://www.gk360.at/api/municipalities/?code=andau

    Counties

    Explore information related to counties, including retrieving all counties or municipalities within a specific county. For example:

    https://www.gk360.at/api/counties/?action=all
    https://www.gk360.at/api/counties/?action=municipalities&county=Mattersburg
    https://www.gk360.at/api/counties/?action=municipalities&county=Oberwart

    Federal States

    Access data about federal states, including their counties or all federal states. For example:

    https://www.gk360.at/api/federal-states/?action=all
    https://www.gk360.at/api/federal-states/?action=counties&federal_state=Burgenland
    https://www.gk360.at/api/federal-states/?action=counties&federal_state=Vorarlberg
